超人归来，要看就要去影院

发表于 2006-07-24 更新于 2020-02-19

超人归来-豆瓣链接

先前看过介绍，这部电影我本来想到IMAX去看。结果看到它在北美票房的虎头蛇尾，连美国人都不给他面子，我也就不掏腰包了。于是买碟回来看，果然是一部名副其实的名不副实的片子。

超人在这部片子里就是一个超级搬运工，从搬人，搬车，搬大铁球到搬飞机，搬游艇，最后还搬了一个小行星。力气是大人是强，但这样老是硬上也没啥意思啊。

后来，被人家小光头找到了弱点（那绿绿的物质叫什么来着），一刀把超人捅到海里面。好不容易被前女友救起来，又赶回去跟人家打。总觉得这次的故事过于紧凑，从路易斯误上游艇开始，到坏人的阴谋被挫败前后不过几个小时。人家精心策划的阴谋被超人一会会儿就解决了，时间跨度小，打赢了也没什么爽的。

还有就是路易斯，本来我一看她已经结婚生子了十分兴奋，好一个痛苦的情感挣扎。但是到后来，什么“超人受到情感挫折”，就他妈带着她飞一下，她就重拾旧爱了，地球人在情场上也完全不是对手嘛。在后来还要好，为了超人火种的传承，大概也为了续集更精彩，那小孩也变成超人的了，这感情真是太没劲了。

然而到最后路易斯和她新男友怎样也没交待，那个坏蛋光头也没被抓住。总之这就是一部不了了之的片子了，续集里面再说吧。

鉴于这是一部情节感情都平平的电影，最好的观赏方式还是去imax看。至少还能戴眼镜享受20分钟的3d巨爽视觉体验，看过的人都说好，我没钱那就不去了。
