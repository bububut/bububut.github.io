探寻泳池之旅

发表于 2007-07-30 更新于 2020-02-19

这是我第一次使用live writer来写，之前觉得它的界面很丑陋，功能很弱。现在轮到我自己来做这样一个东西的时候，发觉能做成这样已经很完善了。

话说我某日去交大游泳，发觉泳池的情况每日愈下，人多水脏的，于是我萌生出游清晨6：15一场的念头。于是在一个温暖的夏日清晨，沐浴着和煦的阳光，我骑车在空荡荡的大街上，心情像被周围的温度加热升空的氢气球。当我终于赶到交大，面对这个寄托着我今夏清凉锻炼加艳遇美梦的碧蓝泳池之时，两滴热泪浸润了我大脚趾旁的老茧。

关于我如何在六点半的泳池里被猛男推搡、被艳阳暴晒、被老婆婆爱抚、被污水灌肠的情形就暂且按下不表。我妈在了解了情况之后当机立断要帮我另觅一家泳池。其实离我们家最近的就是上海游泳馆了，但我妈很果断的说那里必然很贵，妈妈先带你去几个妈妈小时候游的地方，要是不行了我们再去万体馆。。。。

于是我们便踏上了探访历史上著名的游泳馆之路。据说那些馆都在位育中学附近，我很奇怪的是我怎么总来也没有印象。

首先，我们来到了位于乌鲁木齐路上的上海游泳协会暨上海游泳业余学校，哇，都学校了还会没游泳？确实没有，人家说我们专心培训，现在已经不对外开放泳池拉，不过他们说位于复兴路上的上海跳水池还能游哦。这恰巧又跟我妈的另一段泳池记忆相吻合，她说当年上海跳水池可是上海数一数二的大泳池哦。

随即我们又匆匆赶到了复兴路，果然这里有“上海跳水池”的大招牌，我惭愧的想道我在这里读了那么久书居然不知道这里有个闻名遐迩的游泳池。。。。这时我妈已经开始看张贴出来的开放时间和价目表了，并且分析了我什么时候来最恰当。但就在此时，我敏锐的发现了问题的关键。这个贴在游泳池招牌旁边的海报写的居然是网球场开放时间。。。。

最后我妈只好问门口的值班大叔：“这里游泳池现在能游泳吗？”

那个大叔摇摇头说“不游了，关门了。”“啊？他晚上不开门啊，什么时候关门的啊？”此时大叔很潇洒的用手比出一把枪的造型

“八点关门啊？”

“关了八年了。。。”

这时我才恍然想起，在我风华正茂的初中时代，曾经来到过此地踢球，还遇到了不良少年；而早在那时，游泳池已经被改建成了网球场

爱哭的菩萨是不是泥菩萨

发表于 2006-08-10 更新于 2020-02-19

最近李明千叫我找以前每天记事情的小本子，好搞他的什么博物馆。我从一堆行将就木的高三复习材料里面把它挖了出来。翻看时我发现以前抄的一段话，忘记当时是怎么想的了，反正现在看起来值得探讨。我是这样写的

菩萨悲心犹如雪聚，雪聚见日则皆融消，菩萨悲心见苦众生，悲心雪聚放眼中流泪。
一者见修功德人，以爱敬故，为之堕泪；二者见苦恼众生无功德者，以悲悯故，为之堕泪；三者修大施时，悲喜踊跃，亦复堕泪。计菩萨堕泪以来，多四大海水。

提婆罗菩萨造《大丈夫论》

原来菩萨也是这样爱哭的，博大的悲悯之怀让他流尽天下眼泪。但关键是菩萨为什么会哭。被情感左右然后流下眼泪，其实是我们凡夫俗子的作风，陷于情感之流的话怎么会变成菩萨呢。又或者说菩萨的眼泪，产生眼泪的情感，其实并不能左右他的行为，也就是说，菩萨并非真的喜爱修功德之人而厌恶行恶事之徒，因为他本来就没有爱恶欲憎。

看上去是个两难的问题，我想啊想，发现原来就是相的问题。有爱有恨会笑会哭的菩萨是人相的菩萨，菩萨本来是无所谓相的，但世人看到的菩萨却有男相女相寿者相众生相。而情感是构成这些相的基础，因为人与人的交流必须基于共通的情感，所以菩萨也就会哭个不停了。

而且继续追问人相菩萨所流的泪是否真诚也变得没意义了，因为对于眼前的菩萨人，它必定是真诚的。而对于那个无相超脱的菩萨，就没有真诚那回事儿了。须菩提肯定会说：菩萨流泪，实非流泪，是名为泪。

游泳见闻

发表于 2006-07-27 更新于 2020-02-19

交大泳池的深处栖居着声音人。他成天潜泳，透窥女孩的身体，窃听别人谈话。

最近声音人给自己买了新的潜水眼镜，为了看清楚那个老呆在浅水区的女孩。他并不觉得女孩有什么与众不同，只是喜欢看她撞到别人时啊啊大叫的样子，看她跟着教练夸张地做抱水的动作，再看看她用自创的自由泳动作游了一米之后沾沾自喜地笑啊笑。于是声音人的每日生活就显得不那么无趣了。

然后的某一天，当声音人越发觉得想要天天看到女孩时，他决定对她说我爱你。他并不太清楚别的人是不是这样说的，因为没什么人在泳池里说过，但他觉得自己非这么说不可。于是他开始苦恼，要怎么得到这三个字。

根据百科全书记载，声音人是不能自己说话的。他们只能捕获别人说出的字词，然后重新组合变成自己的话。而他们赖以生存的食物也是别人的声音。所以当他们截到别人说的话时，可以吃掉以填饱肚子，或是留着以后说出口。

这个声音人稍有不同，他只能得到别人在水中说的话，于是他的词汇就少得可怜。除去已经吃掉的，他现在储备着的字只有我，年，了，可，团，日。这几个字翻来覆去最多只能组成一句粗话。于是他成天竖着耳朵，渴望得到需要的字。

日子过得很快，声音人的感情尽管不断增长，却得不到表达，他开始憋得慌了。突然一天，一个小胖子在玩闹时大叫“你是李莫愁”，当时他的头正被按到水下。声音人终于得到了“你”。还有一次，有个人在水下说“哎呀”，他得到了这两个字，却将他们都吃了。他不想用妥协的方法对女孩说“我哎你”或是“我矮你”，他要毫不含糊的表白。

但谁会在水下说爱呢。于是声音人开始痛苦，他拒绝进食，拒绝移动，甚至不再看那个女孩。女孩依旧每天游来游去，泳姿日渐成熟，尽管她已深深介入声音人的生活，但她的生活里不论有没有他都是一样快乐的。想到这一点，他吃掉一个日字。

终于有一天，当我被教练残忍地推进深水区，利用求生的本能来学习踩水时，我见到了声音人。他的眼神忧郁绝望，我很不忍心，张口说了说“爱”。我没有听到自己说的这个字，知道它已被声音人捕获了。于是我奋力踩水，浮出水面，再次离开了这个故事。

他终于能说我爱你了。他要游到女孩身边去说，但现在声音人的行动却格外吃力，50米的距离变成很远，一路上他不断吃掉其他的字以补充力气。就当他快到达的时候，铃响了，今天的时间到了。女孩快快地离开了泳池，她还要赶去上班。声音人失望地想，明天就等在浅水区，省得浪费时间。

只是他不知道，今天是游泳班的最后一天。从此以后，声音人每天等着女孩来游泳，等那件白白的胸前绣花的泳衣出现，等她的笑声再响起，却总是等来一群装腔作势愚不可及的男女泳客。于是他又回到深水区，回到以前潜泳偷窥窃听的无趣日子。

然后他把那三个字也吞进了肚子，就像其他毫无意义的字一样。

会写字有啥用

发表于 2006-05-12 更新于 2020-02-19

以前在某人设计的传单上看到过这么一句话：写作吧它能证明你的存在。当时就觉得特别无趣，似乎这种话小时候就看到过很多遍，既老土又自以为是。不料数个月之后我就对这句话敬佩的五体投地。

也不知道从何时起我对文字有一种莫名的崇拜，有可能是中了博尔赫斯的毒。这不是说什么我很喜欢写作拉很喜欢用语言表达自己的感情拉之类的屁话，而是对于文字能够作为一种与外无关，自给自足，在本体论意义上成立的系统的崇敬，如同敬畏自然一般。

说文字与外无关也许令人疑惑，没有世界文字何以依附？没有人类文字何以产生？但想一下，当我们说“电脑“这个词的时候，我们并不需要有一台真实的电脑在身边，也就是说，当一个词语被创造之后，他就与这个真实世界脱离了关系，自我独立的存在着。而所有这些打从历史的开端起就不断增加的文字，通过其互涉性构成了一个独立于我们身处的世界的庞大结构，或者说就是另一个世界。

而我们必须借助于那个世界才能够存在。这就回到了那句曾被我摒弃的话，但是存在主义却把它作为至理名言。海德格尔说此在现身的三个阶段：情绪、领会、言谈。关键是言谈，只有从言谈你才能确定你是什么。因为我们永远生活在可能性之中，而每当你说出一句话、作出一件事、或是写下一段文字，你就确定了一种可能，由此展开自我，现身于这个世界。

我们能逃开文字，不说话不写字地存在于世吗？我没试过，也许可以，但请注意，人是赋形的动物这句话不是白说的。为世界赋予形式似乎是我们的本能，与生俱来，自然而然的。这意味着当你进行思考时，完全自发的运用了形式，也就是用符号指称世界。那便是广义的语言，即使你从不知道语言是什么。

好了我有些神志不清了不说了。不过结论就是，当你说出一句话的时候，你决定了这句话，这句话同时也决定了你。听上去有些问题，以后再慢慢想吧
