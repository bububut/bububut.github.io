路上逛

发表于 2006-02-15 更新于 2020-02-19

大境路青莲街交接处，坐落着白云观。不须多说什么，只看看凌空的飞檐就叫人奇想联翩。意外地见到它时，我还有种偶遇故人的感觉。

进去点了一炷香，殿内神音绕梁，角落里有专人弹奏道教音乐。两面壁上站了百十个大将军，灯光幻影之下，颇有仙境之韵。我找到了守护1986年的大将军，竟似个儒将，丙寅太岁耿章大将军！

观里工作的人看上去都像些流氓，我不知道如果我有机会，会不会选择到这一片隔世的方圆来工作。

然后又沿上海老街走到城隍庙，这里老的新的中的洋的都融在一起，人一多还像是经过压缩的，就成了后现代的微型景观。像是走到指环王里的城堡中去了，乱走八走来到许愿树前，很多人在往上面扔东西。一个女的铆尽全力，结果动作变形水平扔出，激射到旁边人的头上；还有一个扔的超高，穿过层层枝叶却没挂住，直落到围栏里面的地上，再也捡不回来了。

又看了一些稀奇好玩的东西，我就回家了。公交车一边开一边吱吱呀呀地响着，好似在演奏爵士乐，我就在这乐声中睡着了

电视里的死人

发表于 2006-02-12 更新于 2020-02-19

电视简直是最能让人伤心的东西了。无论剧中的故事是悲欢离合，剧中的人物是幸福微笑，还是承受痛苦，总能激起我们无穷的唏嘘。面对电视，我们所流的眼泪绝不亚于菩萨在面对苦恼众生时流下的。

是不是我们就像永恒不变神道一样，看着电视里古今中外的凡夫俗子来来往往。当最终整部电视剧结束时——这几乎是我最不愿接受的事实了，那些给我们带来欢笑泪水种种感情的人物再也不出现了，此刻的落寞惆怅大家总该都体验过吧。

他们都死翘翘了。如果把死亡定义为只拥有过去而没有将来，那一切籍由艺术而诞生在第二世界的人物们，他们的死期就是作者停止创作之日。好吧，我想我们已经发现了艺术人物的好几种死法。

一是在他的世界里死去。在电视剧里当然可以有人死去，但这种死通常是最不彻底的，可任由作者悔改的。著名的例子就像是福尔莫斯之死，死了十几年又能奇迹般的活过来。

二是作者不写了。此种死法的撤销大多是作者挡不住读者的热情，又执笔续写，或是他老人家死了由别人代写。福尔莫斯的例子在这里又可以用了，很有些作家封笔了就把主角弄死，以示决心，其实反悔还不是一二三的事情。至于代写么，这主角复活了也有一种借尸还魂的感觉。

然后就是很超级的了。这种死法是作者肯定不写了，还没有人央求他再写，也没有热心人代写。这种较彻底的死法大多出现在那些没人看的三流小说主人公身上。

哎，作为一个小说人物也那么惨，由此可知，要想长命百岁，必须活得有声有色，让天上看戏的神仙们喜欢你。要是菩萨没了兴趣，不想再看了，那也就没什么希望了。

每日洗菜报告

发表于 2006-02-10 更新于 2020-02-19

最近听人抱怨说我每天都休息，从不更新。我决定发奋一次。然而天天在家闲着确实无趣，不如为大家记录下我每天的洗菜过程吧。

今天妈妈叫我买青菜回来洗，于是我来到小菜场。摊位上有个女孩正低头在吃午饭。我要了一斤青菜。她挑菜时我一不小心往她衣领里面看了一眼，结果发现她非常之大，心想多吃青菜果然有助身体发育。付钱的时候，我突然又发现她的指甲也非常之长，而且狠狠地在我手上抠了一下，抠掉一块皮，仿佛是对我偷看她的报复。

回到家里开始洗菜，扳开第一棵，哇！一条小青虫！它在我面前欢快地扭动着。我正想捏死它，却又大发慈悲，把它丢进垃圾桶，任它自生自灭。菜全部洗好后，再放水把它们浸好。

回到电脑前准备写SPACE，我突然下了一大跳。刚才那条小青虫居然没有丢掉，它扒住我的袖口死不松手。我一把把它拉下来，接着就丢向窗外。

恰在此时吹过一股狂风，把小虫吹向高空。它从我家7楼窗口往上飞，飞过8楼看到屋里一个小孩在做寒假作业，飞过9楼看到两个女人在打架，飞过10楼吓了一个浇花的老头一大跳………就这样飘来飘去，一直飘到18楼窗口晾着的一对胸罩里面。那胸罩像青菜一样是绿色的，上面还有镂空花纹，就像是菜叶子上被虫子咬出的一个个小洞。那小青虫就在这些洞里快活地钻进钻出。

正当我想看得更仔细一点时，从窗里伸出一只手把胸罩摘进去了。于是我今日的细菜过程也全部终结了。不过此时我又突发奇想，假如那个卖菜的小姑娘就是这对胸罩的主人，而当她下一次热情地以乳沟迎接顾客时，那小虫又从她胸口掉回到青菜堆里。他岂不是完成了一次奇妙的回归的历险。
