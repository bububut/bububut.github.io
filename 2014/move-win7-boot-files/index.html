将 Win7 引导程序装到另一块硬盘上

发表于 2014-09-07 更新于 2020-02-19

如果原本在一块硬盘上装有 windows，在另一块硬盘上安装新系统的时候，还是会使用原来硬盘上的引导程序。如果想把原来硬盘进行格式化，windows 的磁盘管理器是不允许的。此时需要把引导程序转移到新的硬盘上。

磁盘管理器里显示的磁盘属性有这么几个是和引导启动相关的：

系统：这次启动是这个盘上的引导程序引导的。
启动：启动的是这个盘上的 windows。
活动：可以用这个盘引导 windows启动。

在 windows 里可以使用 Visual BCD Editor 将引导程序装到另一个硬盘上。具体方法见这里。

如果因为什么问题导致引导程序不能用了，需要使用 windows 安装盘的修复功能。如果在修复的时候找不到现在已经安装的 windows 系统，还需要另行 google 一下解决方法。
